6.2 Descriptions

The interactive part of our project was built with D3 on Observable, which is the official web tool for creating and sharing D3 code. We randomly select 1/100 of the original data-set which was collected from 10/28/2021 to 11/04/2021 within several specific time-slots. Based on the LLN, the newly organized data-set with features user_id,created_at,follower_count,sentiment_score,text,sentiment can still reflect the true distribution of our sentiment analysis. You can find the data here under the data folder of our repo.

6.3 Instruction for using the map

The plot is initialized with the sentiment analysis we get after NLP process with no default follower group and comment search specified. The y-axis gives the sentiment score of each comment with color blue reflecting positive attitudes and red reflecting negative.

  • User can hover the mouse over a circle to see user id, comments he make, and sentiment score in detail.

  • User can search for exact words in the text comments made by Twitter users. Such like entering Tesla in the search bar and click go will gives several data circles in the plot with more positive than negative.

  • User can filter the comments based on the number of followers of the user. This function can be used together with the search function introduced above to plot a customized distribution of comments.

Notes:

  • The x-axis indicate the date value which is not exactly continuous because our data-sets are crawled and collected mostly during evening of a day.
